Теоретическое введение

Уравнение гармонического осциллятора

$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = f(t)$$

где: - β — коэффициент затухания - ω0 — собственная частота - f(t) — внешняя сила

Преобразование в систему первого порядка

Вводим переменные: - x1 = x (смещение) - x2 = ẋ (скорость)

Получаем систему: $$\begin{cases} \dot{x_1} = x_2 \\ \dot{x_2} = -\omega_0^2 x_1 - 2\beta x_2 + f(t) \end{cases}$$

Фазовый портрет

Фазовая траектория — кривая на плоскости (x, ẋ), описывающая эволюцию системы для конкретных начальных условий.

Фазовый портрет — совокупность фазовых траекторий для различных начальных условий.
