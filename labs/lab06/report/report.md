# Введение

Рассмотрим простейшую модель эпидемии. Предположим, что некая популяция,
состоящая из N особей (считаем, что популяция изолирована), подразделяется на
три группы:

- **S(t)** — восприимчивые к болезни, но пока здоровые особи;
- **I(t)** — инфицированные особи, которые также являются распространителями инфекции;
- **R(t)** — здоровые особи с иммунитетом к болезни.

До того, как число заболевших не превышает критического значения I*,
считаем, что все больные изолированы и не заражают здоровых. Когда I(t) > I*,
инфицированные способны заражать восприимчивых к болезни особей.

# Мой вариант

**Номер варианта: 56**

**Параметры модели:**
- $N = 4973$ — общая численность населения;
- $I(0) = 49$ — начальное количество инфицированных;
- $R(0) = 19$ — начальное количество здоровых с иммунитетом;
- $S(0) = N - I(0) - R(0) = 4973 - 49 - 19 = 4905$;
- $\alpha = 0.01$ — коэффициент заболеваемости;
- $\beta = 0.02$ — коэффициент выздоровления;
- $I^* = 100$ — критическое значение.

# Теоретический анализ

## Математическая модель

Скорость изменения числа $S(t)$ меняется по закону:

$$
\frac{dS}{dt} = 
\begin{cases}
-\alpha \dfrac{S I}{I^*}, & \text{если } I(t) > I^* \\[10pt]
0, & \text{если } I(t) \leq I^*
\end{cases}
$$

Скорость изменения числа инфицированных особей:

$$
\frac{dI}{dt} = 
\begin{cases}
\alpha \dfrac{S I}{I^*} - \beta I, & \text{если } I(t) > I^* \\[10pt]
-\beta I, & \text{если } I(t) \leq I^*
\end{cases}
$$

Скорость изменения выздоравливающих особей:

$$
\frac{dR}{dt} = \beta I
$$

## Пороговое значение

Критическое значение $I^* = 100$ является порогом:
- При $I(0) \leq I^*$ больные изолированы, эпидемия не распространяется;
- При $I(0) > I^*$ инфекция распространяется, возникает вспышка эпидемии.

# Результаты моделирования

## Случай 1: I(0) ≤ I* (больные изолированы)

При $I(0) = 49 \leq I^* = 100$ больные изолированы, инфекция не распространяется.
Система уравнений принимает вид:

$$
\begin{cases}
\dfrac{dS}{dt} = 0 \\[10pt]
\dfrac{dI}{dt} = -\beta I \\[10pt]
\dfrac{dR}{dt} = \beta I
\end{cases}
$$

Начальные условия: $S(0) = 4905$, $I(0) = 49$, $R(0) = 19$.

![Динамика популяций для случая 1 (I(0) ≤ I*)](../project/plots/epidemic_case1.png){#fig:case1 width=100%}

**Анализ:**
- Количество восприимчивых $S(t)$ остаётся постоянным ($S(t) = 4905$);
- Количество инфицированных $I(t)$ экспоненциально убывает;
- Количество выздоровевших $R(t)$ растёт и стремится к $I(0) + R(0) = 68$;
- Эпидемия затухает без массового распространения.

## Случай 2: I(0) > I* (инфекция распространяется)

При $I(0) = 150 > I^* = 100$ инфекция распространяется. Система уравнений принимает вид:

$$
\begin{cases}
\dfrac{dS}{dt} = -\alpha \dfrac{S I}{I^*} \\[10pt]
\dfrac{dI}{dt} = \alpha \dfrac{S I}{I^*} - \beta I \\[10pt]
\dfrac{dR}{dt} = \beta I
\end{cases}
$$

Начальные условия: $S(0) = 4973 - 150 - 19 = 4804$, $I(0) = 150$, $R(0) = 19$.

![Динамика популяций для случая 2 (I(0) > I*)](../project/plots/epidemic_case2.png){#fig:case2 width=100%}

**Анализ:**
- Количество восприимчивых $S(t)$ убывает по мере распространения инфекции;
- Количество инфицированных $I(t)$ сначала растёт, достигает пика, затем убывает;
- Количество выздоровевших $R(t)$ растёт и выходит на плато;
- Наблюдается чётко выраженный пик заболеваемости.

## Сравнительный анализ

| Показатель | Случай 1 (I(0)=49) | Случай 2 (I(0)=150) |
|------------|---------------------|----------------------|
| Максимум I(t) | 49 (убывает) | $\approx 120$ |
| Время пика | — | $\approx 15$ ед. времени |
| Конечное S | $\approx 4905$ | $\approx 4650$ |
| Конечное I | $\approx 0$ | $\approx 0$ |
| Конечное R | $\approx 68$ | $\approx 323$ |

# Анализ результатов

## Случай 1 (I(0) ≤ I*)

При начальном количестве инфицированных $I(0)=49$, не превышающем критическое значение $I^*=100$, больные считаются изолированными. Инфекция не передаётся здоровым, поэтому:
- Число восприимчивых $S(t)$ остаётся постоянным;
- Инфицированные выздоравливают с постоянной скоростью $\beta$;
- Эпидемия затухает естественным путём.

## Случай 2 (I(0) > I*)

При начальном количестве инфицированных $I(0)=150$, превышающем критическое значение $I^*=100$, инфекция начинает распространяться:
- На начальном этапе $I(t)$ растёт, так как скорость заражения превышает скорость выздоровления;
- Достигается пик заболеваемости, когда $\alpha \frac{S I}{I^*} = \beta I$;
- Затем $I(t)$ убывает, так как количество восприимчивых $S(t)$ уменьшается;
- Эпидемия затухает после того, как основная масса населения переболеет.

## Пороговое значение I*

Критическое значение $I^* = 100$ является порогом:
- При $I(0) \leq 100$ эпидемия не распространяется;
- При $I(0) > 100$ возникает вспышка эпидемии.

Это демонстрирует важность раннего выявления и изоляции заболевших для предотвращения массового распространения инфекции.
